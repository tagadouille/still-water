# Readme du projet

## Comment compiler et éxécuter :

#### Compilation :

```bash

$ ./gradlew build
```

#### Exécution :

```bash

$ ./gradlew run
```

#### Lancement des tests :

```bash

$ ./gradlew test
```

## Fonctionnalité implémentés :

- Algorithme de calcul de gradient améliorer avec méthode des mailles
- Optimisation multithreader sur les sons et les mises à jours du modèles
- Interface graphique avec affichage des informations de la parties, les curseurs, les particules et etc 60 fois par secondes
- Application des règles de déplacement sur une population de cellules
- Ajout d'IA semi-détermistes rudimentaires
- Multi-joueur local
- Ajout de musiques et de possibilité de joué des sons
- Ajout d'un éditeur de niveau
- Chargement des niveaux depuis des fichiers au format JSON
- Ajout de deux possibilités de finir la partie : être le dernier en lice ou bien être le joueur avec le plus de cellules à la fin du temps imparti
